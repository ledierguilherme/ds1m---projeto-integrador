FPOO

Java

Como um dos conteúdos propostos no curso de "Desenvolvimento de Sistemas", Java é um exemplo de linguagem de programação orientada a objetos, utilizando-a, é possível criarmos aplicações com propósitos diversos, mas que no geral estão ligados à automação de tarefas (criar aplicativos usados em desktops, manipular o back-end).

Eclipse IDE

O Eclispe é uma IDE usada para desenvolver aplicações em Java, porém suporta várias outras linguagens a partir de plugins como C/C++, PHP, ColdFusion, Python, Scala e Kotin. Assim como em todas as profissões, necessitamos de ferramentas para alcançar o devido objetivo, com programação não é diferente. Utilizamos recursos como o "Eclipse IDE", que nos permite programar de maneira mais ágil e efeciente agilizando o processo no geral.

LIMA

Linguagem de Marcação

Em informática, uma linguagem de marcação, ou markup, é um conjunto de sinais e códigos aplicados a um texto ou a dados para definir a sua configuração. A marcação não aparece no trabalho final.

VS Code

O Visual Studio Code é um editor de código-fonte desenvolvido pela Microsoft para Windows, Linux e macOS. Ele inclui suporte para depuração, controle de versionamento Git incorporado, realce de sintaxe, complementação inteligente de código, snippets e refatoração de código..

SOP

Sistemas Operacionais

Tem como objetivo proporcionar o entendimento dos fundamentos técnicos e científicos relativos a estrutura, funcionamento, instalação, configuração e operação de sistemas operacionais de código aberto e fechado. Sistema operativo ou operacional é um programa ou um conjunto de programas cuja função é gerenciar os recursos do sistema, fornecendo uma interface entre o computador e o usuário ou utilizador.

HARE

Em hardware e redes aprendemos sobre os IPs válidos,
brodcast, classe ABC. E aprendemos como utilizar esses IPs usando o prompt de comando.

Aprendemos sobre as utilidades das peças do computador como placa mãe, gabinete, bateria,
ventoinhas, HD, Memória, e muitos outros hardwares.

Aprendemos a conversão de bit, decimal, hexadecimal e octal. A conversão de decimal para octal
(ou seja da base 10 para a base 8),
consiste em dividir progressivamente o valor decimal por 8, obtendo-se um resultado e um resto.
